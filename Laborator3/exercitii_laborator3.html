Exercitii JavaScript

Acestea sunt exercitiile pentru laboratorul 3:

  1. Creati o pagina HTML care afiseaza mesaje folosind alert(), console.log() si document.write().
  2. Declarati variabile pentru nume student, anul de studiu si media si afisati-le in consola.
  3. Scrieti o functie care calculeaza suma si produsul a doua numere.
  4. Creati un array cu 10 numere si utilizati forEach si map.
  5. Creati un obiect car cu proprietatile brand, model si an.
